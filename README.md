## Technologies

Project is created with:

-   React 18
-   Gatsby 5
-   Typescript 4
-   Styled Components 5

## Setup

To run this project, install it locally using npm:

```
$ git clone https://github.com/blvcku/js-design.git
$ cd ./js-design
$ npm install
$ npm start
```
